> **Status:** Steps not yet written. Use [CrowdStrike Falcon Windows sensor deployment docs](https://docs.crowdstrike.com/r/en-US/6tn7yp7b/a33s97dy) in the meantime.

## Overview

Deploy the CrowdStrike Falcon sensor to Windows endpoints using Microsoft Intune (Endpoint Manager). This method packages the sensor as a Win32 app in Intune, enabling deployment to both Azure AD-joined and hybrid-joined devices, including those outside the corporate network.
