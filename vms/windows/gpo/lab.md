## Overview

Deploy the CrowdStrike Falcon sensor to Windows endpoints using Active Directory Group Policy Objects (GPO). This method leverages GPO software installation or startup scripts to push the sensor MSI package to domain-joined machines, providing centralized deployment without requiring additional third-party tooling.
