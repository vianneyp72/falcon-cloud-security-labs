> **Status:** Steps not yet written. Use [CrowdStrike Falcon Operator GitHub](https://github.com/CrowdStrike/falcon-operator) in the meantime.

## Overview

Deploy the CrowdStrike Falcon sensor using the Falcon Operator on Red Hat OpenShift. The Falcon Operator integrates with OpenShift's OperatorHub and handles the additional security context constraints (SCCs) required by OpenShift, managing the full lifecycle of Falcon sensor components via custom resources.
