## Overview

Deploy the CrowdStrike Falcon sensor using the Falcon Operator on any standard Kubernetes cluster. The Falcon Operator is a Kubernetes operator that manages the lifecycle of Falcon sensor components (DaemonSet sensor, sidecar injector, and image analyzer) via custom resources, simplifying deployment and upgrades.
